# 기획 검수 - 다키스트 슬롯던전

## 치명 리스크
- 리스크: 영입/강화 체감이 약하면 슬롯 결과가 단순 운빨로 느껴질 수 있음.
  - 대응: 보상 적용 직후 스탯/로그/전투 결과에 즉시 반영되도록 유지.

## 기획 정합성 점검
- 코어 루프(회전 → 해석 → 보상 → 진행)가 일관적임.
- 수집 + 로그라이트 상호작용이 유의미함(신규 영웅이 룬 풀/행동 다양성에 직접 영향).
- 전투 불능 영웅 룬 제외 + 전술 재분배 규칙으로 이중 페널티를 완화함.

## UX 리스크
- `844x390`에서 패널이 많으면 시선 분산 가능.
  - 대응: 전장/룬 입력 우선, 사이드 패널은 압축 유지.
- 작은 화면에서 보상 모달 가독성 저하 가능.
  - 대응: 모달 높이 제한 + 본문 스크롤 + 최상단 레이어 고정.

## 기술 리스크
- 애니메이션 레이어 과다 시 프레임 저하 가능.
  - 대응: `transform/opacity` 중심의 짧은 애니메이션만 사용.
- 파티 변동 시 룬 매핑 불일치 가능.
  - 대응: 매 스핀마다 현재 출전/생존 상태로 룬 풀 재계산.

## 판정
- 진행 가능(치명적 차단 이슈 없음).
