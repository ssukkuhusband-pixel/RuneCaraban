# FX 규칙 - 다키스트 슬롯던전

## 시각 방향
- 다크 네온 톤 기반(레퍼런스 기획서 스타일)
- 어두운 배경 + 골드/핑크/청록/퍼플 액센트 대비
- 보조 레퍼런스(`/Users/dongjun/Downloads/index.html`)의 짧은 하이라이트 모션(`shimmer`, `slideUp`, `glow`) 채택

## 전투 피드백
- 행동 유닛: 짧은 방향 이동 + 강조 링
- 피격 유닛: 플래시 펄스 + 충격 리플
- 행동자 ↔ 대상 사이 투사체 트레일
- 피해/회복/보호막 플로팅 숫자 표시
- 적 의도 배지(아이콘/명칭/예상 피해) 상시 표시
- 보스 격노 진입 시 카드 강조(색/펄스) 적용
- 궁극기 발동 시 전용 트레일/임팩트와 유닛 하이라이트 적용
- 정예 적 카드 톤 차별화로 난도 신호 강화
- 유물 획득 시 로그/사이드 패널 즉시 갱신으로 보상 인지 강화

## 슬롯 피드백
- 회전 중 릴 흔들림(약 700ms)
- 결과 고정 시 짧은 글로우
- 고정 직후 자동 해석 즉시 시작

## 보상 피드백
- 보상 모달은 어두운 백드롭 위 최상단으로 등장
- 카드 진입 시 순차 슬라이드업, 호버/탭 시 쉬머 하이라이트
- 카드 탭 시 리프트(상승) + 선택 글로우 효과
- 선택 확정 시 모달 닫힘 + 파티/특성 즉시 갱신

## 성능 제약
- `transform/opacity` 중심 애니메이션만 사용
- 히트 단위 연출은 짧게 유지(이벤트당 300ms 이하 권장)
- 무거운 필터/장시간 반복 백그라운드 애니메이션 지양
