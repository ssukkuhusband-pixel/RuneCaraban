# UI/UX 블루프린트 - 룬 캐러밴

## 레이아웃 의도
- 전장을 화면의 시각 중심으로 유지한다.
- 슬롯 입력 영역은 항상 보이고 다른 UI와 겹치지 않게 유지한다.
- 진행도와 수집 상태는 한눈에 읽히게 배치한다.

## 화면 구조 (가로)
1. 상단 상태 바
   - 노드 진행/타입, 파티 체력, 연계 배율, 턴 보너스
2. 중앙 메인 영역
   - 좌측: 전장(아군 vs 적군, 행동/피격 강조, 적 의도 배지 + 궁극기 게이지)
   - 우측: 압축 정보 패널(출전 영웅, 런 특성, 유물, 메타)
3. 하단 입력 바
   - 슬롯 릴 3개 + 회전 버튼

## 핵심 화면 상태
- 시작: 타이틀 + 짧은 설명 + 런 시작 버튼
- 전투: HUD + 전장 + 로그
- 보상: 3지선다 카드 모달
- 종료: 클리어/실패 요약 + 재도전

## 정보 우선순위
- 1순위: 누가 행동하고 누가 맞았는지
- 2순위: 룬 결과, 턴 보너스, 적 의도
- 2.5순위: 궁극기 충전/준비 상태
- 2.6순위: 정예/휴식 노드 여부
- 3순위: 파티/특성 상세 정보
- 4순위: 상세 로그 텍스트

## 가독성 규칙
- `844x390` 기준 패널 겹침 금지
- 모달 본문은 필요 시 내부 스크롤
- 유닛 카드 텍스트는 카드 경계 밖으로 넘치지 않음
- 사이드 패널은 내부 스크롤로 처리하고 메인 레이아웃을 밀지 않음

## 인터랙션 규칙
- 메인 CTA: `회전`
- 보상 선택: 카드 단일 탭
- 우측 상단: 일시정지/재시작
